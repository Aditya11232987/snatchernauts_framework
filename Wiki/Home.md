# Snatchernauts Framework Documentation

**The Revolutionary Ren'Py Framework That Changes Everything**

_Create visual novels that feel alive — Transform passive reading into immersive exploration_

Welcome to the comprehensive documentation for the Snatchernauts Framework. This manual guides you through creating exploration-rich, tactile gameplay experiences that made classics like _Snatcher_ and _Policenauts_ legendary. The framework transforms traditional visual novel storytelling by adding structured room-based exploration, contextual interactions, and cinematic effects.

## 🎯 **What Makes Snatchernauts Different**

### ✨ **Framework Features**
- **Pixel-Perfect Interaction** — No more clunky rectangular hitboxes
- **Unity/Godot Quality Visual Effects** — CRT shaders, bloom, letterboxing, film grain
- **Multi-Input Support** — Mouse, keyboard, and gamepad out of the box
- **Professional Architecture** — Clean, extensible, maintainable code
- **Production-Ready Tooling** — Built-in debugging, automated builds, CI/CD

### 🎮 **Perfect For These Game Types**
- **Detective/Mystery Games** — Pixel-perfect evidence discovery
- **Sci-Fi Adventures** — Retro-futuristic atmospheres with authentic CRT effects
- **Point-and-Click Adventures** — LucasArts-style exploration with modern storytelling
- **Interactive Fiction** — Rich, explorable environments that respond to curiosity
